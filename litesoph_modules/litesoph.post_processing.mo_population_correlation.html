litesoph.post_processing.mo_population_correlation.moocc_correlation_plot module

Code to create contour plot given 3D data. The data is smoothed by a Gaussian and plotted using an inferno color-scale. The following parameters are required as command line input fname -> Name of data file. Data is expected as a 3-column data where the second parameter

is iterated first.

na -> Number of values of first parameter nr -> Number of values of second parameter nev -> Number of divisions for the parameters in the plot sig -> The half-width of gaussian used for smoothing
